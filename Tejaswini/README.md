# Titanic Dataset Analysis using Pandas

## Objective
Analyze Titanic passenger data using Pandas.

## Dataset
Titanic Dataset

## Tools Used
- Python
- Pandas

## Analysis Performed
- Loaded dataset
- Viewed first 5 rows
- Generated summary statistics
- Analyzed survival counts
- Compared survival by gender
- Compared survival by passenger class

## Key Insights

1. Not all passengers survived.
2. Female passengers had a higher survival rate than males.
3. First-class passengers had a higher survival rate compared to other classes.
4. Age and fare varied significantly among passengers.
5. The dataset contains passenger demographic and travel information.
