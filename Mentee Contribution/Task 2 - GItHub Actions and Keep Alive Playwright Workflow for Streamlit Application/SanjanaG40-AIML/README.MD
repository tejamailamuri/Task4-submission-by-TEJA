# Task 2 - Streamlit Keep Alive Workflow

## Objective

Prevent Streamlit Cloud application from sleeping by automatically sending requests using GitHub Actions.

## Streamlit Application

https://proglangpredictor.streamlit.app/

## Technologies Used

- Python
- GitHub Actions
- Requests

## Workflow

1. GitHub Actions runs every 6 hours.
2. Python script sends request to Streamlit app.
3. Streamlit stays active.
